Previous Next ? HTML Symbol Entities HTML symbols like mathematical operators, arrows, technical symbols and shapes, are not present on a normal keyboard. To add these symbols to an HTML page, you can use the HTML entity name. If no entity name exists, you can use the entity number. If the character does not have an entity name, you can use a decimal (or hexadecimal) reference. If you use an HTML entity name or a hexadecimal number, the character will always display correctly. This is independent of what character set (encoding) your page uses!
